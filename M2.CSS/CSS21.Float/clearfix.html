CSS ClearFix

Khi 1 thẻ HTML cao hơn thẻ mẹ của nó và nó được Float left hoặc right
=> THì nó sẽ trượt ra khỏi thẻ
=> Thẻ mẹ không còn chứa thẻ con nữa
Vậy để giải quyết vấn đề này có 2 cách phổ biến
Cách 1: Đặt thẻ mẹ thuộc tính Overflow:hidden Cách này có nhược điểm
Cách 2: Sử dụng Class ClearFix và đặt class này cho thẻ mẹ
.clearfix {
overflow: auto;
}
.clearfix::after {
content: "";
clear: both;
display: table;
}
