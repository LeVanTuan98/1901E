CSS Inline-Block

Inline-Block là sự kết hợp giữa Inline và Block

Chú ý: Các thẻ Inline như span, a thì đặt width và height trong CSS không có tác dụng

Con lai của Block và Inline

Nó sẽ lấy đặc tính hiển thị trên 1 hàng của Inline

Nó lấy đặc tính đặt được width và height trong css của Block

Khi nào cần sử dụng display:inline-block;

Cần đặt height và width cho các thẻ inline mà không muốn làm mất đặc tính hiển thị trên 1 hạng của Inline
