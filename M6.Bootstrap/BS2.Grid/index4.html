CSS Responsive Grid Class

  1. col-* áp dụng cho màn hình Mobile ( từ 0 đến 575px) VD: col-1, col-2, col-3,...,col-12
  2. col-sm-* áp dụng cho màn hình máy tính bảng như Ipad (từ 576px đến 767px) VD: col-sm-1, col-sm-2, col-sm-3,...,col-sm-12
  3. col-md-* áp dụng cho màn hình Laptop (từ 768px đến 991px) VD: col-md-1, col-md-2,col-md-3,...,col-md-12
  4. col-lg-* áp dụng cho màn hình PC (từ 992px đến 1199px) VD: col-lg-1, col-lg-2, col-lg-3, ..., col-lg-12
  5. col-xl-* áp dụng cho màn hình siêu lớn ( từ 1200px đến vô cực ) VD: col-xl-1, col-xl-2, col-xl-3,...,col-xl-12
