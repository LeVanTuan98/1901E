Phần tử Block và Inline

BLOCK (Khối) là các phần tử HTML khi tạo ra nó sở hữu 1 hàng riêng biệt. Có thể tùy chỉnh kích thước của Block VD:<div>,<h1>,<h2>,<p>,<ul>,...

INLINE (Hàng) là các phần tử HTML khi tạo ra nó có thể đứng trên cùng 1 hàng.VD :<span>,<strong>,<a>,...
